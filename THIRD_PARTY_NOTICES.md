# Third-party notices

This project is a lightweight graphical wrapper around third-party tools. The app code in this repository is MIT licensed, but bundled dependencies keep their own licences.

## yt-dlp

If you distribute `yt-dlp.exe` with the installer, include the licence and notices required by the yt-dlp project.

## FFmpeg

If you distribute `ffmpeg.exe` and `ffprobe.exe` with the installer, make sure your FFmpeg build and distribution method comply with the licence terms of that build. `scripts\get_dependencies.ps1` downloads a GPL Windows build from `yt-dlp/FFmpeg-Builds` by default.

## User responsibility

This app does not grant permission to download copyrighted or restricted content. Users should only save content they own, have permission to download, or are otherwise legally allowed to save.
